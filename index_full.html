    FAQ (Kurz)

    Wie funktioniert das Impact-Network?

    Jede:r Käufer:in erhält einen persönlichen Code. Empfehlungen werden transparent gezählt; es gibt keine Geldprämien, sondern sichtbare Wirkung (z. B. „4 Familien versorgt“).

    Wird meine Spende nachverfolgt?

    Ja. Jede Spende wird einem Projekt zugeordnet und auf der Map + im Dashboard angezeigt.

    Kann ich das Gerät draußen betreiben?

    Ja, Solar/Akku ermöglicht Off-Grid-Betrieb, 24/7 nutzbar.
